01
AND (&)
Both bits must be 1. Used to check/clear bits.
n & 1 — check if odd; n & (n-1) — clear lowest set bit
02
OR (|)
At least one bit is 1. Used to set bits.
n | (1 << k) — set bit k
03
XOR (^)
Bits differ. Self-inverse: a^a=0, a^0=a.
a ^ b ^ b == a — find unique element
04
Shift (<<, >>)
Left shift = ×2. Right shift = ÷2.
n << 1 = n*2; n >> 1 = n//2

# Common bit tricks
n = 12  # 1100 in binary

n & 1          # check if odd → 0 (even)
n & (n-1)    # clear lowest set bit → 8
n | (n-1)    # set all bits below MSB
n ^ n         # XOR with self → 0
n >> 1        # divide by 2 → 6
n << 1        # multiply by 2 → 24

# Count set bits (Brian Kernighan)
def count_bits(n):
    count = 0
    while n:
        n &= n - 1  # clears lowest set bit
        count += 1
    return count
